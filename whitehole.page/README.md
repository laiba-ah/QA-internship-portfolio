## Project Overview

This repository contains the Quality Assurance (QA) testing documentation for the White Hole website. The testing was performed to evaluate the website's functionality, usability, user experience, and responsiveness across different sections and devices.

## Testing Performed

The following testing activities were completed:

- Functional Testing
- UI/UX Testing
- Responsive Testing (Mobile & Tablet)
- Bug Reporting
- Test Case Design
- Test Planning

## Documents Included

This repository contains the following documents:

1. Test Plan
2. Functional Test Cases
3. Bug Report
4. UI/UX Testing Report
5. Responsive Testing Observations (included in the UI/UX report)
